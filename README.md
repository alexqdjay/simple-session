# Simple Session
> A Distributed Session Frame (Java)  
一个简化`SpringSession`的分布式Session框架, 基于Java, 旨在简单好用

## quick start

### 1. Spring boot

**Step1.** Add `@EnableSimpleSession` on your Main Class


```Java
@SpringBootApplication
@EnableSimpleSession
public class MainRun {
	...
}
```

**Step2.** Config your properties
